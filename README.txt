Udacity Memory game

Project Overview
The Memory Game Project is all about demonstrating your mastery of HTML, CSS, and JavaScript.

Game Rules

The game randomly shuffles the cards. A user wins once all cards have successfully been matched.
The game displays a star rating (from 1 to at least 3) that reflects the player's performance.
When the player starts a game, a timer starts.

How to play

Click a card to flip over and reveal the icon. Two cards can be flipped over to reveal the icon.
Flip over two hidden cards at a time to locate the ones that match.
